# CLI Blackjack


## Deliverables
> As a user i can
* player draw
* dealer draw
* randomize card draw
* shuffle deck
* reset deck
* over 21 ends game
* ace = 1 or 11
* dealer must hit below value 16
* stand on value 17
* ask to play again
* counting player wins
* username input
* determine winner
